cdec is a decoder, aligner, and learning framework for statistical machine translation and similar structured prediction models.

Features

  • Translation and alignment modeling and inference based on finite-state transducers and synchronous context-free grammars.
  • Implementations of several parameter learning algorithms.
  • Fast, extensible C++ implementation and Python interface (pycdec).
  • Modular architecture that separates search space construction, rescoring, and inference.
